1. Décompresser le dossier appelé "projetTelioPaul",

2. Sortir du dossier compressé,

3. Télécharger XAMPP sur votre ordinateur via le lien suivant :
    https://www.apachefriends.org/fr/download.html

4. Ouvrir XAMPP,

5. Sur XAMPP ouvrir le serveur Apache et MySQL, 

6. Aller dans le dossier Documents/xamp/htdocs sur Windows ou /Applications/XAMPP/xamppfiles/htdocs sur MacOS, créez-y un dossier "index",

7. Y insérer le dossier phpproject qui se trouve dans le dossier "ProjetTelioPaul",

8. Ouvrir phpmyadmin "http://127.0.0.1/phpmyadmin/" sur Windows ou "http://127.0.0.1/phpmyadmin/index.php?route=/" sur MacOS

9. Créer une nouvelle base de données en cliquant sur le bouton éponyme à gauche,

10. Y mettre le nom "NetflixPrimePlus",

11. Ne pas créer de table mais cliquer sur "importer" et choisissez le fichier "NetflixPrimePlus.sql.gz" qui se trouve dans le dossier appelé "projetTelioPaul", 

12. Pour ouvrir le site allez sur : "http://127.0.0.1/index/projetphp/projetphp/"sur Windows ou "http://localhost/phpFiles/projetphp/login.php" sur MacOS

13. Cliquer sur projetphp 

14. Et découvrez notre site (cliquez sur les images pour accéder aux informations).
